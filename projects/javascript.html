Itmay not seem amazing, but this example shows how javascript isnt just useful for creating websites. Just look at the code for the prime calculations: 

function primeSieve(limit){
	flags = []
	for (i = 0; i < limit ; i++){
		flags.push(true)
	}
	flags[0] = false
	flags[1] = false
	primes = []
	for (index = 0; index < limit; index ++){
		if (flags[index] == true) {
			primes.push(index)
			for (n = index * index ; n < limit ; n += index){
					flags[n] = false
			}
		}
	}
	return primes
}
The code is almost idential to the same function written in python: 
def primeSieve(limit):
	flags = [True] * limit
	flags[0] = flags[1] = False
	primes = []
	for (index, flag) in enumerate(flags):
		if flag:
			primes.append(index)
			for n in range(index * index, limit, index):
				flags[n] = False
	return primes
But why you ask? Well this is just one aplication of javascripts functionality. It out performed the near idential python code here time-wise and it allowed for a nice graphical representation...
