Projects

I've attempted a variety of different projects, physical and software, and only some of which I have written about here.

My main coding languages are Python, Javascript and C (mainly with Arduinos). I also know a tiny bit of Java. Finally, I consider myself quite profficient in OpenSCAD which is CAD software that allows you to describe 3d models through programming.

For writing algorithms, solving problems, scientific computing, networking and generally any quick project, Python is always my go-to. The quick syntax and the interpreter that it offers makes it really easy to write code that is elegant, succinct and generally pythonic. However, as Python is highly terminal based, it is difficult to post my projects on this website and even if I explainied the code, it wouldn't be too interesting. For that reason I haven't written up any posts here, but you can see some of my work on my GitHub and take a read through my StackOverflow answers if you are interested (at the time of writing I have answered nearly 400 questions which are nearly all Python related). There are links to these in the top right.

On the other hand, projects written in JavaScript lend themselves to posts and explanations much more. If you hover over the projects tab at the top, you will be able to select JavaScript and see some of my Projects. Most of those posts (once you click through on the images) have links at the top which will take you to a place where you can view them. I won't go in to what I use JavaScript for here, but you can read this about page I wrote on them which explains why I love coding in it.

Finally, my Arduino projects. As with the JavaScript ones, you can access these through the drop down in the site banner. Some of these projects are also interfaced with Python (as mentioned in their descriptions) so you can see another use of that language.
