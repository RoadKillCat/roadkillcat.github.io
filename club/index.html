Arduino Club

The "/club" sector of this website will be used to log the progress of the "Arduino Club" which I was granted some money by the school to run. This page will be used to track our expenditures.

Our First Project: a classic snake game on a matrix of LEDs.

To get used to how the group would work, we decided to begin with something that wouldn't be too challenging, but would create something cool.

We used the MAX7219 8x8 LED matrix for the display and just wired it up loosely to start with as a prototype so that we could focus our attention on the software (programming the game) rather than the hardware.

After a couple of weeks we managed to get the core design of the snake game working well using some little push switches for inputs.

The Arduino sketch for this can be seen over on github.

We were prototyping with an Arduino Mega and a breadboard connected with some jumpers:

Now that we had the thing working, we decided that it would be nice to have a finnished product that was one device. For this, we all needed to be able to solder (to make a proper controller) and this would be a valuable skill for us in the future, so I gave a little demonstration, and everyone picked it up straight away.

We were now ready to get the components together to build the final product - the main being a more compact matrix. We brought an unsoldered one off ebay quite cheaply, but it required us to solder it together including the miniscule surface-mount chip.

And here are some pictures of our work soldering it together:

However, after soldering the components together for this, we realised that to make the project as small as possible, it would make sense to just use the larger matrix as this had enough space to mount the Arduino Nano we would be using and the controller board. So even though this was a valuable lesson on soldering, we ended up not using it!

Next, we made the controller board by soldering some push buttons to some copper stripboard and wiring this directly to the Nano, along with the LED matrix.

And that was pretty much it! We uploaded the sketch to the Nano that we had already coded for the Mega (the code is compatible between board types in this case) and that was it. Here are some photos of the finnished product, and finally a video of it in action.
